# WA1 — A+ Standard Comparison Chart copy

This is text-only. Amazon's Standard Comparison Chart pulls each book's cover image and title automatically from the ASIN — you just supply the chart heading, the four ASINs, and the row data below.

In KDP, go to the A+ Content editor → Add Module → **Standard Comparison Chart**. The module gives you up to 6 product columns and 10 comparison rows. We're using 4 columns and 5 rows.

## Alt text (under 80 chars each — won't trip Amazon's limit)

Standard Comparison Chart doesn't take alt text on the auto-pulled covers. The fields you'll see are the chart heading and the cell text — neither has alt-text input.

---

## Optional: shrink to 3 columns if any row feels cramped

The Standard chart can collapse to 3 columns gracefully. If the WA2 column feels like a different beast (slice-of-life vs the action stack), one workable variant is to drop WA2 and run all three Undercover Archmage books — re-framing the chart as "If you liked The WarMage of Arkley, here's the rest of the Ravenwick Chronicle." Row 3 becomes the differentiator: trope per book.

Either configuration works. Up to you.
